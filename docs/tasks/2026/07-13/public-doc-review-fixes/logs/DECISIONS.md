# 결정 기록

**2026-07-13**

- **Background:** GitHub 이슈·PR 양식은 다국어 지원과 유지보수 비용이 있지만 현재 사용 우선순위가 낮다.
- **Decision:** Project Legibility의 issue templates, issue config와 pull request template을 모두 제거한다.
- **Why:** 사용하지 않는 기여 표면을 보완하기보다 없애는 편이 현재 운영과 문서 소유권에 맞다.
- **Impact:** GitHub는 blank issue와 기본 PR 작성 화면을 사용하며 별도 양식 계약을 유지하지 않는다.
