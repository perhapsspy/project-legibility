# Decisions

**2026-07-23**

- **Background:** 제품 계약에 이번 교정에서 나온 판단·개입 원칙을 별도 절로 추가하면서 두 스킬의 변경이 Project Legibility 전체의 판단 철학과 공개 정체성 개편으로 확장됐다.
- **Decision:** 앞선 결정의 제품 전역 적용 부분을 철회한다. 감지와 탐색 방식은 `purpose-fit-design`과 `structure-first` 정본 안에서만 조정하고, 제품 계약에는 기존 제품 약속·구성과 달라진 호출 경계만 둔다.
- **Why:** 이번 작업은 두 스킬의 행동 교정이며, 모든 전문 스킬과 제품 정체성을 함께 바꿀 근거는 없다.
- **Impact:** `PRODUCT.md`의 전역 판단 원칙과 반복 설명을 제거하고, README와 플러그인 명세는 기존 제품 중심을 보존하면서 영향을 받은 두 스킬 설명만 갱신한다.
